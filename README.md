## Getting Started / Installing

Fork & clone the repo, then...
Install gems, migrate & seed the db, start the rails server.

1. `$ rails bundle`
2. `$ rails db:migrate`
3. `$ rails db:seed`
4. `$ rails s`

Go to `localhost:3000` in a web broswer to view.

### Prerequisites

It's a [Ruby on Rails](http://rubyonrails.org/) project.
